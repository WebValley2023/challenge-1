RELATIONSHIP BETWEEN LOMBARDIA AND RIVA 

Correlation_Riva_Lombardia.ipynb
This code makes heatmaps and a Linear Regression Analysis between Ozone in Riva and in Lombardia from 2021-12-10 to 2022-12-10

source of Lombardia data:"..\\Lombardia\\lomb_tot.csv"

source of Riva data:"..\\Riva\\DataSet_Riva_garda.csv"


Lombardia_Night_winter.ipynb
This code makes a scatter plot for Riva and one for Lombardia to visualize the direction of the wind during the winter

start_date="2021-12-01"
end_date="2022-3-01"

source of Lombardia data:"..\\Lombardia\\lomb_tot.csv"

source of Riva data:"..\\Riva\\Riva_windDir.csv"


Only_Night.ipynb
This code makes a Linear Regression Analysis between Ozone in Riva and in Lombardia from 2021-12-10 to 2022-12-10 considering only the night from 19:00 to 7:00

source of Lombardia data:"..\\Lombardia\\lomb_tot.csv"

source of Riva data:"..\\Riva\\DataSet_Riva_garda.csv"
