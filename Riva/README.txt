RIVA ANALYSIS 

source of dataset: "DataSet_Riva_garda.csv"

Correlation_Riva.ipynb
Calculates correlation matrices and generates visualizations (heatmaps and scatterplot matrices) to explore the relationships between variables in the datasets 

daily.ipynb
Makes a line graph for each day from 2019-11-15 to 2020-11-15 to compare Ozone and Solar Radiation.

Line_Graph_Riva.ipynb
Generates plots to visualize all the pollutants and weather phenomena measurements from 2013 to 2020

Regression_Riva.ipynb
This script makes a Linear Regression Analysis between each pollutant and weather phenomena in the dataset.
